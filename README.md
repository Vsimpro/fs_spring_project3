# Project 3
Fullstack course, Spring 2023.
## Social media site that allows users to post, edit & delete status updates

### How to run:
Inside `fs_spring_project2` folder 
Frontend:
```bash
cd frontend
npm start
```

Backend:
```bash
cd backend

npm install cors
npm install dotenv
npm install express
npm install mongoose
npm install cookie-parser

node index.js
```
